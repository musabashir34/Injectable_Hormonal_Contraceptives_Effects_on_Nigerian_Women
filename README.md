# Injectable_Hormonal_Contraceptives_Effects_on_Nigerian_Women

This repository contains 
1. the R script to generate figures and data (Analysis.R),
2. Rmarkdown file for generating a manuscript (Article.Rmd)
3. bibliography file (mybibfile.bib) and
4. citation style language (the-pan-african-medical-journal.csl)

All for an article titled "Effects of Common Injectable Hormonal Contraceptives on Selected Indices of Central Adiposity among Hausa Women in Kano Metropolis, Nigeria" submitted to the Pan African Medical Journal.
The R script has been sourced at the top of the Rmarkdwon file which you can knit directly to run the script, create the figures and tables directory from which the markdown script reads the necessary figures and tables.
