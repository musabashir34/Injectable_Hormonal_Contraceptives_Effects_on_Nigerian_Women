# Injectable_Hormonal_Contraceptives_Effects_on_Nigerian_Women

this is the manuscript (manuscript.docx), R script to generate figures and data (Analysis.R), Rmarkdown file for generating the manuscript (Article.Rmd) and bibliography file for the references used in the article (mybibfile.bib). The R script has been sourced at the top of the Rmarkdwon file which you can knit directly to run the script, create the figures and tables directory from which the markdown script reads the necessary figures and tables. 
The article was submitted to the Pan African Medical Journal
